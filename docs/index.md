# Projeto Todo List MVC

Bem-vindo à documentação oficial do projeto Todo List. Esta aplicação foi desenvolvida em **Python** utilizando o padrão de arquitetura **MVC (Model-View-Controller)** e uma estrutura de **Monorepo**.

## Funcionalidades Principais
*   **Cadastrar Tarefas**: Permite adicionar novas atividades com lembretes.
*   **Remover Tarefas**: Exclusão de itens via identificador único.
*   **Lembretes**: Suporte para anotações de tempo ou observações importantes.

## Como Executar
Para rodar o projeto localmente, utilize o comando na raiz:
`py main.py`
